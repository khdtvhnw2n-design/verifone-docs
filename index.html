P400 Installasjonsguide

Installasjon

Åpne lokket, sett inn kabel og sett på lokket igjen. Koble til strømkabel og nettverkskabel. Det tar ca. 5 sekunder før displayet lyser og opptil 1 minutt før programmet har startet. Pass på at du kobler nettverkskabelen til riktig nettverksuttak (merket ETH).

Meny og navigasjon

Terminalen har en berøringsskjerm, og alle menyvalgene kan utføres ved å trykke på skjermen. I enkelte menyer og lister må du kanskje bla (˄/˅) for å vise alle alternativer. All inntasting avsluttes med grønn tast.

For å få tilgang til terminalmenyen, trykk 4 og 6 samtidig. Terminalen vil be om et passord. Tast inn 1234 og bekreft med grønn tast.

Oppsett av terminal

Når du mottar terminalen må den settes opp på riktig kommunikasjon for at den skal få forbindelse og motta oppsett for ditt brukersted. Når terminalen er koblet til nettverket så vil parametere og brukeroppsett lastes ned automatisk. Det kan ta opptil 10 minutter før terminalen er operativ.

Tilkobling via Ethernet

  1. Trykk 4 + 6 og skriv inn passord 1234.
  2. Velg Administrationsmeny → Settings → Communications → Configuration.
  3. Velg Network Interface → Ethernet → DHCP → Nei.
  4. Tast inn IP-adresse med 12 siffer (f.eks. 192.168.001.001).
  5. Skriv inn nettverksmasken og gateway.
  6. Velg Lagre og bekreft med OK.
  7. Terminalen spør "Bruk grensesnittinnstillinger" → Trykk Ja.
  8. Trykk på den røde tasten til du forlater menysystemet.

Tilkobling via WLAN

  1. Åpne terminalens meny.
  2. Velg Administrationsmeny → Settings → Communications → Configuration.
  3. Velg Network Interface → WiFi → WiFi Scan.
  4. Velg nettverk → Legg til.
  5. Sett Autostart til “Yes”.
  6. Velg Pre-Shared Key (PSK), skriv inn passord og bekreft med grønn tast.
  7. Velg “Save” → “OK”.
  8. Velg “Yes” for Apply settings on interface → Interface started → OK → Rød tast for å avslutte.

Konfigurasjon mot kasse

  1. Trykk 4 + 6 og skriv inn passord 1234.
  2. Velg Administrationsmeny → Settings → ECR → Set ECR Address.
  3. Skriv inn kassens IP-adresse.
  4. Velg Sett ECR Port og tast inn 9600.
  5. Trykk på den røde tasten til du forlater menysystemet.

Når terminalen har kontakt med kassen vises kun ”Verifone”. Terminalen er da klar til bruk.
